#

# Requirement
1. Laravel
2. MySql
3. Postman
 
# Installation
```
1. Clone this repo
2. Run command composer install in this folder
3. Set up .env
4. Run command php artisan migrate
5. Run command php artisan key:generate
6. Run command php artisan serve
```
